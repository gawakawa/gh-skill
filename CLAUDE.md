# CLAUDE.md

## Purpose

A sandbox for experimenting with managing Agent Skills via the GitHub CLI:
https://github.blog/changelog/2026-04-16-manage-agent-skills-with-github-cli/
The goal is to exercise the create/fetch/manage skill commands and observe how they behave.

## Commands

- `nix fmt` - Format code
- `nix flake check` - Run checks (format, lint)
- `nix build` - Build the project
